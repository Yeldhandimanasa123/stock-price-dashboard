#  AI Stock Price Predictor

An AI-powered stock prediction dashboard using LSTM and real-time data.

##  Features
- Predict next 7 days stock prices
- Interactive charts with indicators (MA20, Bollinger Bands)
- RSI (Relative Strength Index)
- Download predictions as CSV
- Clean modern UI (React + Tailwind)

## Tech Stack
- Frontend: React, Tailwind CSS, Recharts
- Backend: Flask, Python
- ML Model: LSTM (TensorFlow/Keras)
- Data: Yahoo Finance API
